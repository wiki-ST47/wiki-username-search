Username Search Tool

This is a tool that uses the Toolforge database replicas to find username that match a certain pattern. It is common for sockpuppeters to use some common structure to their username, so this tool can be used to find those accounts.

The "search query" field uses a largely perl-compatible regular expression. ^ represents the beginning of the string, and $ represents the end. So to find all users whose usernames start with "ST47", you could use "^ST47.*$" or more simply "^ST47".

Logging in via OAuth allows you to get more results. For performance reasons, anonymous users have a limited number of results.

Maintainer and Source

This tool is written in Python using Django and Pywikibot, and is maintained by ST47. Source code is available on GitHub.

Bookmarks

Please bookmark the Homepage for normal usage of this tool. However, all of the parameters as passed as "GET" parameters - as part of the URL - so if you want to bookmark a specific query and check back periodically, in case of repeat sockmasters, you can do so.

Options

Path to the wiki
This must be the domain name of the wiki you intend to use, formatted like "en.wikipedia.org", with no "https://" or "/wiki/". English Wikipedia is the default.
Search query
This is the RegEx we are searching for, in perl-compatible format.
Number of users to search
How many users to return in total.
